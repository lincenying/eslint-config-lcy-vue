# eslint-config-vue

A set of opinionated ESLint (http://eslint.org) rules (all rules included) tailored for Vue.js projects

Usage

- npm install --save-dev eslint-config-lcy-vue
- create a file named .eslintrc in your project:
```
{
  "extends": "lcy-vue"
  // Your overrides...
}
```
